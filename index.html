  • ASR Training with Synthetic Data From VC Systems

    • This project detects the effectiveness of a Cycle- GAN based voice converter on speaker identification (SID) systems and an automated speech recognition (ASR) system. The main idea is to improve ASR training with synthetic data from VC systems, which can successfully mimic target speakers' styles. Our experiment shows that when we use part of an open-source Chinese Mandarin speech(Aishell) as the source samples and the VC model is trained using LibriSpeech. We have already got the result that this method could improve the error rate from 0.1245 to 0.1097 on the dataset Aishell, which is a considerable result.

  • Setup of a Scanning Microwave Impedance Microscope

    Course Work/ Small Projects

  • FootprintIDPlus: New Person Identification Method Using Wavelet Coefficients and CNN Network

    • FootprintID is the task that uses footstep-induced floor vibration to realize person identification. In this work, we use wavelet coefficients (0~133Hz, morse wavelet) as feature vectors and use a simplified "AlexNet" to train and test and get 90% step accuracy and 99% trace accuracy in this task. (Current Method: 83% step accuracy and 97% trace accuracy)

  • Machine Learning && Deep Learning (Summer School Program in MIT)

    • Our work is to build a feed forward convolutional neural network model that detects COVID-19 positive patients from negative patients and visualize our model's predictions using Class Activation Mappings (CAM) to gain insight into where our model is attending when making predictions. And finally, we got 81.2% accuracy on the test dataset and 99.8% accuracy on the training dataset after specifying the initial value of the weights in the model (the COVID-19 Chest X-Ray dataset from Cohen J.P., Morrison, P., and Dao L ). And in the second project, it can tell obviously through figures that our model successfully visualizes where our model is attending when making predictions. (the result can be above)
